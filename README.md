#### 1、如何打开文件

```bash
# 克隆仓库
git clone git@github.com:Emma-Deve/shouxiu_client.git

# 进入shouxiu_client文件夹
cd shouxiu_client

# 安装依赖
npm install
# 或者
yarn add

# 启动项目：http://localhost:3800
npm run start
# 或者
yarn start


```
